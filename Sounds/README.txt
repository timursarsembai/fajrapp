Папка для аудиофайлов азана
===========================

Добавьте сюда следующие файлы:

1. azan.mp3 - Обычный азан (для всех намазов кроме Фаджр)
2. azan_fajr.mp3 - Утренний азан (для намаза Фаджр)

Рекомендуемые источники для скачивания:
- https://www.islamcan.com/audio/adhan/
- https://www.assabile.com/adhan

Примечание: Утренний азан отличается от обычного - 
в нём есть дополнительная фраза "Ас-салату хайрун мин ан-наум" 
(Молитва лучше сна).
