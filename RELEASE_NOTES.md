## 🎨 Новые возможности

- **Темы оформления** — светлая и тёмная темы для всех окон приложения
- **Настройка прозрачности** — управление прозрачностью виджета от 30% до 100% с шагом 1%
- **Улучшенный интерфейс настроек** — боковое меню с вкладками для лучшей организации (Язык, Местоположение, Метод расчёта, Смещения, Уведомления, Внешний вид)
- **Улучшенный поиск города** — показывает до 10 результатов с указанием региона и страны при вводе названия

## 🔧 Улучшения

- Переработан дизайн окна настроек с боковой панелью навигации
- Автоматическая адаптация цветов текста и фона для обеих тем
- Улучшенная читаемость текста в разных темах (координаты, смещения времени)
- Централизованное управление темами через ThemeManager
- Правильные цвета при наведении на вкладки в обеих темах
- Корректное отображение текста невыбранных вкладок

## 📋 Требования

- Windows 10/11
- [.NET 8.0 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/8.0)

## 📥 Установка

### Portable версия
1. Скачайте `FajrApp-Portable.zip` из раздела Assets ниже
2. Распакуйте в любую папку
3. Запустите `FajrApp.exe`
4. Настройте своё местоположение и наслаждайтесь!

### Требуется .NET Runtime?
Если приложение не запускается, установите [.NET 8.0 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/8.0/runtime)
